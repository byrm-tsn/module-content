## Moving



```
WHEN FLAG CLICKED
	SET ROTATION STYLE LEFT-RIGHT
	POINT IN DIRECTION 90
	WAIT 1 SECONDS
	MOVE 20 STEPS
	WAIT 1 SECONDS
	POINT IN DIRECTION -90
	WAIT 1 SECONDS
	MOVE 20 STEPS
STOP ALL
```
